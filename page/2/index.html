数学学习

发表于 2022-06-17 更新于 2022-06-18 分类于 数学 Valine：
本文字数： 4.5k 阅读时长 ≈ 4 分钟

写这份文档的目的

读研究生不到两个月，我已经见到周围5个人觉得自己数学不够用（包括我自己），想提升自己的数学能力。因为我之前辅修过数学专业，所以觉得自己有这方面的经验，于是便好为人师地去给别人推荐各种各样的教材和课程。但我在交流的过程中发现如下问题：

  • 我在向别人推荐数学教材与课程时，更多的是在炫耀自己的表面的学识（装X）
  • 我现在还没达到教人做事的水平。目前唯一可以做的，是讲述自己的经验，提出自己的建议，供别人进行判断
  • 我自己也需要一个提升数学能力的计划

成功的人各有各的经验，遇到的坑却往往惊人的相似。在这篇文档中，我将重点阐述自己遇到的坑。有意思的是，坑是经验的，这意味着：如果我没踩过，我可能就不觉得这是坑（手动滑稽）。踩坑可以说是一种反例式的实践的学习方法，但有时非常耗时。所以，若能从我踩过的坑中吸取经验，避免踩坑；或者之前身处坑中而不自觉，看到这篇文档稍有醒悟，那我写这篇文档的目的就达到了。当然，这篇文档也不是用踩过的坑反向教人做事，判断与选择的权利依旧掌握在读者手上。

自己学习数学的经历

我会以时间顺序讲述自己学数学的过程，其中评注内容我会用分割线与主体部分隔开，这样可以不影响叙述的完整性。

确定辅修数学的原因

我在大二上学期开始尝试自学一些数学学科，其中有常微分方程（MIT OCW 18.03），线性代数（MIT OCW 18.06）等，但在大三上学期我决定辅修数学，原因如下：

  • 我想系统地学习数学知识，而不是停留在兴趣的层面上
  • 通过听课、作业、交流、考试，我可以对某一方面的数学知识有一个比较好的把握；而自学只有听课环节，所以即使花费很多精力看完所有课程录像，我还是觉得没能真正掌握这门课教授的知识
  • 选课会施加一种外部的压力，让我学到自闭时也不敢放松（比如学习实变函数和泛函分析）。这一点和第一条很像，兴趣和意识到某件事的重要性当然很重要，但是对于我来说，它们难以让我克服重重困难
  • 辅修提供一个清晰的可以达到的学习目标，同时有验收指标，比如考试成绩（这很重要。自学的往往缺乏一种对所学课程的自信，我觉得部分是这个原因）
