通过 linux 终端获取网页信息

发表于 2024-04-05 更新于 2024-04-06

在 PT 网站上下载了许多种子, 其中一些种子已经完成做种任务要求了, 我想把这些用来做任务的种子从硬盘上删除, 但是完成任务的种子大概有 50 个, 并且其中一些一直保持活跃的种子我想保留下来接着刷流量. 我想从 Linux 终端中获取这些信息, 然后进行数据处理.

使用 curl, wget

这两个命令可以直接从网站上拉取信息, 或者下载文件, 详细的操作方法可以使用 man 或者 tldr.

我认为这两个命令似乎比较适合从网站上下载文件, 至于得到网页上的信息, 还是有点麻烦.

使用命令行浏览器 lynx, w3m

lynx 卡在登录界面进不去, 但是 w3m 就可以, 不知道为什么…

## 将网页信息导出到本地文件中

目前使用命令 w3m https://tjupt.org, 进入到登录界面. 输入用户名以及密码之后, 使用键盘导航到 H&R 详情界面, 查看全部种子. w3m 提供命令 S 保存当前 buffer 中的所有内容.

我们希望知道的关键信息有

  1. 种子名称. 在方括号当中;
  2. 种子 ID;
  3. 种子大小;
  4. 分享率. 分享率高的种子可以继续做种赚取上传量;
  5. 活跃时间. 活跃时间越接近当前时间, 说明种子可能被分享的几率就越大;
  6. 是否正在做种. 只有考核完成的种子才能够被删除.
