有限单元法基本概念(1)

发表于 2022-07-18 分类于 FEM Valine：
本文字数： 5.2k 阅读时长 ≈ 5 分钟

已知一个数学微分方程, 如何建立它的等效积分形式? 如何证明两者是等效的?

若微分方程\(\boldsymbol{A}(\boldsymbol{\phi}) = \boldsymbol{0}\)在域\(\Omega\)中恒成立, 则下式成立: \[ \int_{\Omega} \boldsymbol{v}^T \boldsymbol{A}(\boldsymbol{\phi}) \mathrm{d} \Omega = 0 \label{eq:equivalent_integration_form} \] 其中 \(\boldsymbol{v}\) 是任意函数. 正是由 \(\boldsymbol{v}\) 的任意性, 才能保证上式与原微分方程是等效的.

证明等效性, 必要性有微分方程在域\(\Omega\)恒等于0, 积分形式自然对任意函数 \(\boldsymbol{v}\) 都成立; 充分性可以通过反证法, 假设微分方程在域 \(\Omega\) 中某点处不等于0, 则总可以构造出 \(\boldsymbol{v}\), 比如在该点处的Delta函数, 使得等效积分形式不等于0.

等效积分形式和等效积分"弱"形式的区别何在? 为什么后者在数值分析中得到更多的应用?

\[ \text{等效积分形式} \stackrel{\text{分部积分}}{\longrightarrow} \text{等效积分"弱"形式} \notag \]

通过分部积分, 原微分方程作用于场变量上的导数阶次降低, 转移到试探函数\(\boldsymbol{v}\)上, 由此降低了近似解的连续性要求, 因此在数值求解中近似函数可选择的空间更大. 例如, 对于一般弹性力学问题, 方程中关于位移场的导数最高阶次为1, 故只需要保证位移场函数\(C_0\)连续. 对于梁相关的问题, 方程中关于挠度的导数最高阶次为2, 故需要保证挠度函数\(C_1\)连续.

不同形式的加权余量法之间的区别何在?

对于微分方程\(\boldsymbol{A}(\boldsymbol{\phi}) = \boldsymbol{0}\), 取近似函数\(\bar{\boldsymbol{\phi}} = \boldsymbol {N}\boldsymbol{a}\), 代入微分方程当中, 会得到残差如下 \[ \boldsymbol{R} = \boldsymbol{A}(\boldsymbol{N} \boldsymbol{a}) \neq \boldsymbol{0} ~\text{on} ~ \Omega \] 加权残值法是等效积分形式的有限形式, 即用有限数量\(n\)个规定的权函数代替任意函数\(\boldsymbol{v}\), 可得 \[ \int_{\Omega} \boldsymbol{W}_{j}^{T} \boldsymbol{A}(\boldsymbol{N} \boldsymbol{a}) \mathrm{d} \Omega = \boldsymbol{0} \quad (j = 1~to~n) \] 不同的加权余量法权函数是不同的. 例如, 配点法的权函数是 Delta 函数, 子域法的权函数是子区域恒为1, 其它区域恒为0的函数.

什么是加权余量的伽辽金方法? 它有什么特点?

伽辽金方法的权函数取自于近似函数序列, 即取 \(\boldsymbol{W}_j = \boldsymbol{N}_j\). 若定义近似解\(\bar{\boldsymbol{\phi}}\)的变分为 \[ \delta \bar{\boldsymbol{\phi}} = \boldsymbol{N}_1 \delta \boldsymbol{a}_1 + \cdots + \boldsymbol{N}_n \delta \boldsymbol{a}_n \] 则伽辽金法可以表示成如下形式: \[ \int_{\Omega} \delta \bar{\boldsymbol{\phi}}^{T} \boldsymbol{A}(\bar{\boldsymbol{\phi}}) \mathrm{d} \Omega = \boldsymbol{0} \] 若算子\(\boldsymbol{A}\)是\(2m\)阶线性自伴随的, 则采用伽辽金方法得到的方程系数矩阵是对称的.

本文字数： 1.8k 阅读时长 ≈ 2 分钟

问题描述

本文档旨在解决的问题是:

  • 如何将自定义样式文件.sty通过 MiKTeX 链接到 TeXstudio 平台?
  • 如何更新自定义宏包?

问题解决

第一个问题

第一个问题有一个比较笨的解决方案: 将所有的.sty文件与自己编写的.tex文件放在同一文件目录下. 但这不符合审美. MiKTeX 也下载了许多宏包, 但并不需要将下载的宏包放在同一文件目录下. 所以, 能否将自定义的.sty文件作为类似于 MiKTeX 自动下载的宏包一样, 在外部链接到自己当前工作目录下?

MiKTeX 官方提供了一种方式: 设置TEXMF root directories. 这篇文档写得"通俗易懂", 但很难实操. 一个写的比较好的回答可参考这个网址:

  • Create a folder somewhere on your computer – with an important exception. It must not be a subfolder of your MiKTeX installation(MiKTeX 安装子目录) (it worked this way in older MiKTeX versions, but was changed for security reasons). Avoid as far as possible spaces in the path, it is a potential risk for malfunction. For use with MiKTeX portable the folder must be, of course, on the same drive as the MiKTeX installation, it will be saved as relative path. Here I will use c:\localtexmf.

    This folder must be TDS compliant (see thereto A Directory Structure for TeX Files)(就是我之前提到的TEXMF root directories); MiKTeX will complain, if not. What you need, depends on what you want to install into this directory, at least a tex\latex subtree should be sufficient. You will go safe, if you mimic the structure of the (itself TDS compliant) MiKTEX tree, but binaries can directly go into a subfolder c:\localtexmf\bin.

  • Now, using the GUI:

    1. In the Start Menu go to the MiKTeX entry and open the settings – if you act as admin respectively “Settings (Admin)”, of course. The “MiKTeX Options” window will open.
    2. Go to the “Roots” tab. Click on “Add” and choose c:\localtexmf\. If there are more than one local directory, one can choose the search order with the “Up” and “Down” buttons.
    3. Now the almost most important part: Go to the “General” tab and click there on “Refresh FNDB” (FNDB = File Name Data Base)(MiKTeX4.5, 中文环境下, 直接点击上方工作栏"任务"->"刷新文件名数据库"). In some cases, especially if new fonts are installed, one has to press also the “Update Formats” button.

    If you’d forget this step, all earlier steps would be done to no purpose.(第三步不做, 前面白搞)

I should like to say two things, one intellectual and one moral.

The intellectual thing I should want to say to them, is this: When you are studying any matter, or considering any philosophy, ask yourself only what are the facts and what is the truth that the facts bear out. 复合材料物理场模型

发表于 2022-06-17 更新于 2022-06-22 分类于 复合材料 Valine：
本文字数： 2.2k 阅读时长 ≈ 2 分钟

文档目的

复合材料加工过程至少涉及热学-化学-机械三种物理场的耦合作用, 其中, 热学的场方程遵循 Fourier 传热定律, 而化学和机械的场方程形式依赖于材料的类型. 例如热固性材料的化学场方程需要考虑材料的固化程度, 而热塑性材料则需要考虑其结晶度. 对于碳纤维增强的基体材料, 基体和纤维可能都需要考虑损伤本构模型, 基体材料可能还需要加入黏弹性本构模型.

故本文档旨在整理复合材料除力学本构模型之外的物理场模型.

热力学模型

热传导模型就是经典的 Fourier 方程: \[ \rho C_p \frac{\partial T}{\partial t} = \nabla(\boldsymbol{k} \nabla T) + q_{gen} \label{eq:transient} \] 不同类型的材料, 热源项产生的原因不同. 热固性材料放热是因为固化过程中的交联反应放热; 热塑性材料是因为降温过程中材料结晶放热. 式\(\eqref{eq:transient}\)中热源项可以表示成如下形式: \[ q_{gen} = (1 - V_f) \rho_r H_{tr} R_{r}(\psi, T) \label{eq:heat_source} \] 其中, \(H_{tr}\) 是总反应热, \(\psi\) 表示材料固化或结晶化程度.

化学反应模型

在介绍化学反应模型之前, 先引入一些化学反应中常用的术语和概念.

一些术语

玻璃化转变温度

the cured glass transition temperature (\(T_g\)) of a polymeric material is the temperature at which it changes from a rigid, glassy solid into a softer, semiflexible material.
