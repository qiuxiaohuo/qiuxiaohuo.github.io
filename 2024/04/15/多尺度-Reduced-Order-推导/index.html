多尺度 Reduced Order 推导

发表于 2024-04-15 分类于 力学

多尺度方法中的 Green 函数物理意义

多尺度的单胞问题中, 本构方程为:

\[ \sigma^{\xi}(x) = L^{\xi}(x):\left(\varepsilon^{\xi}(x) - \mu^{\xi}(x)\right) \]

对位移变量进行渐近展开, 得到

\[ u^{\xi}(x) = u^{c}(x) + \xi u^{(1)}(x,y) + O(\xi^2) \]

同样也可以得到应变,应力,本征应变的展开式. 通过应力平衡方程, 以及解的存在唯一条件, 可以得到渐进展开式中不同阶次项之间的关系, 最终可以整理为: \[ \begin{aligned} \varepsilon^{\xi}(x) &= \varepsilon^f(x,y) + \xi \varepsilon^{(1)}(x,y) + O(\xi^2)\\ \sigma^{\xi}(x) &= \sigma^{f}(x,y) + \xi \sigma^{(1)}(x,y) + O(\xi^2)\\ \mu^{\xi}(x) &= \mu^f(x,y) + \xi \mu^{(1)}(x,y) + O(\xi^2) \end{aligned} \]

其中, fina scale 上的变量之间的关系式为:

\[ \varepsilon^f = \underbrace{\nabla_x u^c}_{:=\varepsilon^c} + \nabla_y u^{(1)}\\ \sigma^f = L(y) : \left( \varepsilon^f - \mu^f \right) \]

在多尺度中, 表征单胞内更精细的位移场 \(u^{(1)}\) 是通过如下方式进行构造的:

\[ u_i^{(1)}(x,y) = \varepsilon_{kl}^c(x) H_i^{kl}(y) + \int_{\Theta} \mu_{kl}^f(x,y^{\prime}) h_{i}^{kl}(y,y^{\prime}) \mathrm{d} y^{\prime} \]

根据夹杂问题的推导过程, 由本征应变产生的位移为

\[ u_{i}(y) = -\int L_{jkmn} G_{j,k}^{i}(y-y^{\prime}) \mu_{mn}(y^{\prime}) \mathrm{d} y^{\prime} \]

如果将夹杂问题的 Green 函数记作

\[ h_{i}^{mn}(y,y^{\prime}) = L_{mnjk}(y^{\prime}) \frac{\partial}{\partial y_{k}^{\prime}}G_{j}^{i}(y-y^{\prime}) \]

所以, \(h_{i}^{mn}\) 的含义是, 作用在点 \(y\) 处 i 方向上的单位力所产生的应力场(?).

Reduced Order: Green 函数在 Partition 上的积分

多尺度在单胞上 fine-scale 应力满足平衡方程以及边界条件:

\[ \nabla_{y} \cdot \sigma^{f} = 0, \quad \sigma^{f} \cdot n \in -\# \]

将渐进展开式, 本构方程, 以及关于 \(u_{i}^{(1)}\) 的表达式代入到平衡方程中得到

\[ \left\{ L_{ijkl}(y) \left[ E_{kl}^{mn}(y) \varepsilon_{mn}^c(x) + \int_{\Theta} \mu_{mn}^f(x,y^{\prime}) h_{(k,y_l)}^{mn}(y,y^{\prime}) \mathrm{d} y^{\prime} - \mu_{kl}^f(x,y) \right] \right\}_{,y_i} = 0 \]

线弹性部分的单胞方程为:

\[ \left\{ L_{ijkl}(y) E_{kl}^{mn}(y) \right\}_{,y_i} = 0 \]

本征应变的单胞方程为

\[ \left\{ L_{ijkl}(y) \left[ \int_{\Theta} \mu_{mn}^f(x,y^{\prime}) h_{(k,y_l)}^{mn}(y,y^{\prime}) \mathrm{d} y^{\prime} - \mu_{kl}^f(x,y) \right] \right\}_{,y_i} = 0 \]

将单胞分成若干个 partition: \(\Theta^{(1)}, \Theta^{(2)}, \ldots, \Theta^{(N)}\),并假设每一个 partition 上的本征应变是相同的, 每个 partition 上的本征应变为 \(\mu_{mn}^{(1)}(x), \mu_{mn}^{(1)}(x), \ldots, \mu_{mn}^{(N)}(x)\), 这样得到本征应变在单胞上的分段常值分布:

\[ \mu_{mn}^{f}(x,y) = \mu_{mn}^{(\alpha)}(x) N^{(\alpha)}(y) \quad \text{(使用求和约定)} \]

代入到本征应变单胞方程中, 得到

\[ \mu_{mn}^{(\alpha)}(x)\left\{ L_{ijkl}(y) \left[ \int_{\Theta} h_{(k,y_l)}^{mn}(y,y^{\prime}) N^{(\alpha)}(y^{\prime}) \mathrm{d} y^{\prime} - I_{kl}^{mn}N^{(\alpha)}(y) \right] \right\}_{,y_i} = 0 \]

上式是关于 \(\mu_{mn}^{(\alpha)}(x)\) 的方程组, 且方程组系数独立于方程变量. 这样, 如果方程组的解空间为 \(\mathbb{R}_N\), 也就是对任意 \(\mu_{mn}^{(\alpha)}(x) \in \mathbb{R}_N\), 上述方程恒成立, 则必须要求方程组系数恒等于 0:

\[ \left\{ L_{ijkl}(y) \left[ \int_{\Theta} h_{(k,y_l)}^{mn}(y,y^{\prime}) N^{(\alpha)}(y^{\prime}) \mathrm{d} y^{\prime} - I_{kl}^{mn}N^{(\alpha)}(y) \right] \right\}_{,y_i} = 0, \quad \forall \alpha \]

将 Green 函数在 partition 上的积分记作

\[ P_{kl}^{mn(\alpha)}(y) = \int_{\Theta^{(\alpha)}} h_{(k,y_l)}^{mn}(y,y^{\prime}) N^{(\alpha)}(y^{\prime}) \mathrm{d} y^{\prime} \]

这样就得到关于张量 \(P_{kl}^{mn(\alpha)}\) 的 \(N\) 个方程:

\[ \left\{ L_{ijkl}(y) \left[ P_{kl}^{mn(\alpha)}(y) - I_{kl}^{mn}N^{(\alpha)}(y) \right] \right\}_{,y_i} = 0, \text{in}~ \Theta \quad \alpha=1,2, \ldots,N \]

多尺度得到的单胞应变场与本征应变的关系

只考虑本征应变对 fine-scale 应变的影响, 我们有如下关系式:

\[ \varepsilon_{ij}^{f}(x,y) = \int_{\Theta} h_{(i,y_j)}^{kl}(y,y^{\prime}) \mu_{kl}^{f}(x,y^{\prime}) \mathrm{d} y^{\prime} \]

将 reduced order 分块均匀假设代入上式, 就得到

\[ \varepsilon_{ij}^{f}(x,y) = P_{ij}^{kl(\alpha)}(y) \mu_{kl}^{(\alpha)}(x) \]

在每一个分块 \(\Theta^{(\alpha)}\) 上对 \(\varepsilon^f\) 进行积分平均, 得到 \[ \varepsilon_{ij}^{(\beta)}(x) = \langle P_{ij}^{kl(\alpha)}(y) \rangle_{\Theta^{(\beta)}} \mu_{kl}^{(\alpha)}(x) = P_{ij}^{kl(\beta \alpha)} \mu_{kl}^{(\alpha)}(x) \]
